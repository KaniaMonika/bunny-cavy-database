# Bunny & Cavy Database System

## Overview
This project is a fully functional relational database system and desktop application developed for a custom e-commerce business concept focused on selling dried herbs and herbal blends for rabbits and guinea pigs.

The project was created as part of the Advanced Database Systems module during my BSc (Hons) Computer Science degree at Solent University.

The system combines SQL, SQLite and Python to simulate a real online store environment with:
- inventory management,
- customer accounts,
- order processing,
- business logic automation,
- graphical user interface functionality.

## Installation & Setup

### 1. Clone the repository

```bash
git clone <your-repository-link>
```

### 2. Install required libraries

```bash
pip install bcrypt
```

### 3. Run the application

From the main project folder run:

```bash
python run.py
```
